# 채맵 무료 서버 설치 (구글 앱스 스크립트)

관리자님 Gmail만 있으면 됩니다. 돈 안 들고, 카드 등록도 없어요.
아래 6단계 중 **관리자님이 하실 건 계정에 올려서 주소 2개를 받아오는 것**뿐입니다.

## 1. 스크립트 만들기
1. 컴퓨터 브라우저에서 **script.google.com** 접속 (Gmail 로그인)
2. 왼쪽 위 **+ 새 프로젝트**
3. 이름을 "채맵서버"로 (위쪽 '제목 없는 프로젝트' 클릭해서 변경)

## 2. 코드 붙여넣기
1. 가운데 편집기의 기존 내용(`function myFunction()...`) 전부 지우기
2. `채맵서버.gs` 파일 내용을 **전부 복사해서 붙여넣기**
3. 위쪽 💾(저장) 아이콘 클릭

## 3. 최초 실행 (setup)
1. 편집기 위쪽 함수 선택 칸에서 **setup** 선택
2. **▶ 실행** 클릭
3. "권한 검토" 창 → 본인 Gmail 계정 선택 → "고급" → "채맵서버(안전하지 않음)으로 이동" → "허용"
   (내가 만든 내 스크립트라 안전합니다)
4. 실행이 끝나면 아래 **실행 로그**에 `ADMIN_KEY(관리자 열쇠): xxxxxxxx` 가 나옴 → **이 값을 복사**

## 4. 웹 앱으로 배포
1. 오른쪽 위 **배포 → 새 배포**
2. 톱니바퀴 ⚙ → **웹 앱** 선택
3. 설정: 실행 계정 "나", **액세스 권한 "모든 사용자"**
4. **배포** → 다시 권한 허용
5. 나오는 **웹 앱 URL**(https://script.google.com/macros/s/..../exec) **복사**

## 5. 클로드에게 알려주기
아래 두 개를 채팅에 붙여넣기:
- 웹 앱 URL: `https://script.google.com/macros/s/..../exec`
- ADMIN_KEY: `xxxxxxxx`

## 6. 나머지는 클로드가
앱이 이 서버와 연결되도록 클로드가 코드를 마저 붙여 배포합니다.
그러면 지인이 등록 → 서버에 쌓임 → 관리자님이 승인 → 이름·주소 자동 수집 → 지도에 자동 표시.
