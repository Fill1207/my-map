# 📍 나만의 지도

데이트 · 여행지 · 맛집 · 카페 · 낚시포인트 · 교회를 전국 지도로 정리하는 우리끼리 장소 기록장.

**접속: https://fill1207.github.io/my-map/** — 열려면 비밀번호가 필요합니다 (아는 사람에게 물어보세요).

## 장소 제안하기 (누구나)
앱 오른쪽 위 **✉️ 장소 제안** 버튼 → 카카오맵 링크와 한마디를 적고 **공유** → 관리자에게 카톡으로 전송.
관리자가 확인하면 지도에 올라갑니다. 등록·삭제는 관리자만 할 수 있어요.

## 관리자용: 데이터 수정 방법
장소 원본 데이터(`places.js`)는 개인정보 보호를 위해 **이 저장소에 올라가지 않습니다** (관리자 컴퓨터에만 있음).
사이트에는 암호화된 `places.enc.js`만 올라갑니다.

1. 로컬 `places.js`를 수정 (형식은 파일 상단 주석 참고)
2. `node tools/build.mjs <비밀번호>` 실행 → `places.enc.js` 재생성
3. `places.enc.js` 커밋 후 push → 1~2분 뒤 사이트 반영

## 파일 구성
- `index.html` — 앱 전체 (잠금 화면 + 지도 + 카드 UI)
- `places.enc.js` — 암호화된 장소 데이터 (AES-256-GCM, gzip)
- `koreamap.js` — 대한민국 시·도 경계 지도 (southkorea-maps 오픈소스 기반)
- `tools/build.mjs` — 로컬 원본을 암호화하는 빌드 스크립트

## 데이터 체계
전국을 **8도 + 제주**로 나누고 **시 → 구 → 동** 순서로 정리합니다.
광역시는 가까운 도 그룹에 포함: 서울·인천→경기 / 대전·세종→충남 / 광주→전남 / 대구→경북 / 부산·울산→경남
